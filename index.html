Connecting Local and Remote Repository using GitHub and Git (Push,Pull,Remove)

  1. Open Browser: Type Url GitHub.com then Sign in, Add new Repository & Open Git

  2. Type: Git init -b main

    3. (Initializing default (-b)branch main)


  3. Type: Git add .

    4. (Staging them for commit)


  4. Type: Git commit -m "First Commit"

    5. (Commits tracked changes (-m for message) and prepares them to be pushed to a remote repository)


  5. Type: Git config --global user.name"<username>"

  6. Type: Git config --global user.email"<email>"

    7. (Log in via email and username in Git)


  7. Type: Git remote add origin <REMOTE_URL>

    8. (Set up the new remote)


  8. Type: Git remote -v

    9. (Verifying(-v) remote)


  9. Type: Git push origin main

    10. (Push the changes in your local repository to remote repository you specify)


  10. Type: Git push -f origin main

    11. (Force to push all changes from local to remote)


  11. Type: Git pull origin main

    12. (Pull or Getting updated file from remote repository to local repository)


  12. Type: Git rm --cached <file_name>

    13. (Removing file in remote repo)


  13. Type: Git log --oneline

    14. (Showing all commits in one line)


  14. Type: Git config --global --unset credential.helper

  15. Type: Git config --system --unset credential.helper

    16. (You could also disable use of the Git credential cache using this or Go to Windows Credentials in your Windows)
